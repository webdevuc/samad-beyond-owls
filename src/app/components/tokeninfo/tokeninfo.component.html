Grise Smart Contracts

Grise project is designed with several contacts that work together to capitalize and grow in a sustainable, safe way but at a faster rate than any other system that exists. Phase one of Grise token mechanism will happen in the following manner

Launch event

The launch contract will create 50 slots that will be 4 hours long. During this phase users can send BNB or any other defi token to the contract to reserve grise for the Main liquidity pool.

Circulation Event
